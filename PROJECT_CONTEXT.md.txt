# CAMPUSCONNECT PROJECT DOCUMENTATION

# Project Name

CampusConnect

---

# Project Purpose

CampusConnect is a college social networking platform.

Students can

- Chat with each other
- Buy & Sell products
- Share notes
- Upload posts
- Manage profile
- Connect with classmates

Current Login Flow

User Registers

↓

User Saved in MongoDB

↓

OTP Generated

↓

OTP should be sent by email

↓

User verifies OTP

↓

emailVerified = true

↓

User can Login
